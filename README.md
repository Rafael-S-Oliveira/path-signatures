# pathsignatures 📈

> **Implementação de Assinaturas de Caminho (Path Signatures) e Regressão Lasso em R**

O pacote `pathsignatures` fornece um conjunto de ferramentas robustas para a 
aplicação da teoria de *rough paths* (assinaturas de caminho) na modelagem 
preditiva de séries temporais. Desenvolvido para lidar com dados sequenciais 
complexos (como cenários epidemiológicos e surtos de dengue), o pacote integra 
de assinaturas matemáticas com modelos de *Machine Learning* 
(Lasso e abordagens Bayesianas).
